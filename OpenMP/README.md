# Shared-Memory Parallelism with OpenMP

This folder collects materials for OpenMP lectures.
The plan is as follows:

#### `slides` folder

1. Introduction to parallelism
   *general concepts on parallel computation, shared- and distributes- memory paradigms as hardware and programming paradigms, basic elements of parallel architectures, definition of parallel performance and scalability.*

2. Introduction to OpenMP
   *OpenMP as one possible implementation of shared-memory programming paradigm, its philosophy and historical evolution.*

3. Parallel Regions
   *How to open and manage parallel regions, and nested parallel regions. Shared- and private- data, the threads’ stack. How to specialize execution within a parallel region, how to protect code regions and data locations.*

4. Loops

   *How to taks advantage of OpenMP’s shortcuts to manage loops.*

5. OpenMP Memory model
   *A concise introduction to OpenMP memory model how to synchronize threads directly.*

#### `notes` folder

Here you’ll find the lecture notes, drafted from the transcription of the lectures, and some additional materials
